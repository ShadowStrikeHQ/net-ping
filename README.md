# net-ping
Sends ICMP echo requests to a given host and reports response times. - Focused on Basic network operations

## Install
`git clone https://github.com/ShadowStrikeHQ/net-ping`

## Usage
`./net-ping [params]`

## Parameters
- `-h`: Host to ping
- `-c`: Number of pings
